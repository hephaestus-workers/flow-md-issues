# FlowMD - Issue Tracker

This repository is the public issue tracker for **[FlowMD](https://marketplace.visualstudio.com/items?itemName=hephaestus-workers.flow-md)**, a WYSIWYG Markdown editor for VS Code.

## How to Report

- **Bug Report** - Something isn't working as expected
- **Feature Request** - Suggest a new feature or improvement

Please use the [Issues](https://github.com/hephaestus-workers/flow-md-issues/issues) tab to submit your report.

## Before Submitting

1. Search [existing issues](https://github.com/hephaestus-workers/flow-md-issues/issues) to avoid duplicates
2. Include your VS Code version and OS
3. For bugs, include steps to reproduce
